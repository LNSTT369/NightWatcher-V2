Note on HFT execution: NightWatcher V3's signal routing and order management infrastructure is independent software. The partnership with Richard Kim's firm concerns institutional clearing access — NightWatcher itself does not operate as an HFT system or co-located exchange participant. All trades execute via brokerage API (Alpaca) subject to standard market latency. The "dark pool" signal source type refers to signals derived from dark pool print data, not direct dark pool order routing.
